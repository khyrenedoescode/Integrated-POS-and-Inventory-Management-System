# 🛒 Point of Sale & Inventory Management System

> A modern desktop-based Point of Sale (POS) and Inventory Management System built with C# and Windows Forms, enhanced and customized as part of my software development portfolio.

---

## 👩‍💻 About This Project

This repository showcases my work in **customizing, enhancing, and innovating** an existing Point of Sale and Inventory Management System. Rather than developing the application entirely from scratch, I focused on improving its functionality, refining the user interface, optimizing workflows, and extending several features to create a more polished and user-friendly retail management solution.

This project reflects my experience working with desktop applications, database management, and software customization using C# and Microsoft SQL Server.
